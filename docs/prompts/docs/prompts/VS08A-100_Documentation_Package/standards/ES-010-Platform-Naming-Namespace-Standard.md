# ES-010 — Platform Naming & Namespace Standard

## Purpose
Provide consistent naming across CAP.

## Domain Models
Use PascalCase:
- PlatformApplication
- PlatformApplicationPackage
- Tenant
- TenantWorkspace
- WorkspaceInstallation

## Database
Use snake_case plural table names:
- platform_applications
- platform_application_packages
- tenants
- tenant_workspaces
- workspace_installations
